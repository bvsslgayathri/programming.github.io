HTML LISTS

Html also has great feature of creating lists.Suppose if you want to give the information like the food items you like, you would list them out . This can be done using <li> tag. Now lets see how we can list the data./p>

So, basically lists are of 3 types.

  1. unordered list
  2. ordered list
  3. description list


Let's look into the detailed description. What basically is unordered lists? It can be used to group a set of related items in no particular order. We can understand by this that ordered list is used to group a set of related items in a specific order. Then what is description list? We all are aware of dictionary, description list works similar to dictionary.

Unordered list

An unordered list starts with <ul> and each item starts with <li>

Lists are marked with disc bullets by default.

Example: output

We can also change bullets and have square or a circle or a disc.Lets see an example.

<ul type="square">

<li>water</li>

<li>food</li>

</ul>

The result of using type="square" would give us square bullets.

  • water
  • food

In similar manner we can use circle,disc type bullets.

Ordered lists

In similar way we can create ordered lists simply by using <ol> instead <ul> of

lets see an example

It is clear that list is marked with numbers by default.

As in unordered list we can change numbers to Default-Case Numerals, Upper-Case Numerals, Lower-Case Numerals,Upper-Case Letters, Lower-Case Letters.

<ol type = "1"> - Default-Case Numerals.
< ol type = "I"> - Upper-Case Numerals.
<ol type = "i"> - Lower-Case Numerals.
<ol type = "A"> - Upper-Case Letters.
<ol type = "a"> - Lower-Case Letters.

Lets see an example:

<ol type = "I">

<li>food</li>

<li>water</li>

</ol>

Now result would be

  1. food

  2. water

Similarly you can use other types.

Suppose, if we are using numbers and a case occurs where we have to start with number 5 , what shall we do then!
For such case we can use start attribute.

<ol type = "I" start="5">

<li>food</li>

<li>water</li>

</ol>

See what happens now!

  1. food

  2. water

Description list

Next,HTML Description List or Definition List displays elements in definition form like in dictionary. The <dl>, <dt> and <dd> tags are used to define description list.

The 3 HTML description list tags are given below:

  • <dl> Defines the start of the list
  • <dt> A term
  • <dd> Term definition
  • </dl> Defines the end of the list

Lets see it with example

<dl >
<dt> <b>HTML </b>
</dt>
<dd>This stands for Hyper Text Markup Language</dd>
<dt>
<b>HTTP </b>
</dt>
<dd>This stands for Hyper Text Transfer Protocol</dd>

</dl>

Result would be

HTML
This stands for Hyper Text Markup Language
HTTP
This stands for Hyper Text Transfer Protocol
